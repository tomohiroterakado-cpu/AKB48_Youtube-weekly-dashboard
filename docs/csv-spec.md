# CSV項目仕様

## 必須列

| 正規項目 | 対応する列名 |
|---|---|
| 動画ID | `動画ID`、`コンテンツ` |
| 動画タイトル | `動画タイトル`、`動画のタイトル` |

## 任意列

公開日時、長さ、ユニーク視聴者数、新しい視聴者数、リピーター、視聴回数、総再生時間、チャンネル登録者、登録者増加数、推定収益、平均視聴時間、インプレッション数、CTR、高評価数、コメント追加回数、共有数、平均視聴率、Shorts視聴選択率、スワイプ率。

## 粒度

- `合計` 行: チャンネル全体の週次値。新しい視聴者、リピーター、ユニーク視聴者などの正本。
- 動画行: 動画別指標。空欄を0に変換しない。
- `上位500件の結果を表示しています`: YouTube Studioの説明行なので除外。

チャンネル全体の新規視聴者・リピーターを、動画別に単純合計しません。

## エラー

- 必須列なし: 取込を停止し、想定列名を表示。
- 文字化け記号を検出: UTF-8で再出力するよう表示。
- 不正動画ID: 行をスキップして件数表示。
- 欠損値: 取込可能。欠損項目と件数を警告。
- 同一ファイル: スキップ。
