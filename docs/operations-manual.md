# 毎週の運用マニュアル

## 市場調査レポートの自動反映

AI Directorの週次レポート下部にある「外部環境の示唆」は、Gmailで受け取る件名に `市場調査レポート` を含むメールから作成します。

初回だけ、Apps Scriptのプロジェクト設定でスクリプトプロパティ `DIRECTOR_ADMIN_TOKEN` を追加し、値にAI Directorの管理コードを入力します。さらに `MARKET_REPORT_ALLOWED_SENDERS` を追加し、受信を許可する送信元メールアドレスを入力します。その後、`setupMarketResearchImportTrigger` を1回実行します。

以後は、メール受信後おおむね1時間以内に最新メールを読み込みます。同じメールは重複追加されません。対象期間を本文から特定できない場合、土曜〜金曜の7日間でない場合、または同じ週のCSVが先に反映されていない場合は保存せず、エラーとして残します。修正版など同じ週の別メールは、表示中の内容を勝手に上書きせず、確認待ちとして保存します。

SNS・検索の定量情報がメール本文にない場合は、画面に「データ不足のため判定不可」と表示します。推測によるトレンドは追加されません。

## はじめに: 管理モードにログイン

閲覧だけの場合はログイン不要です。CSVの取込や動画情報の更新をする前に、画面右上の **管理モードにログイン** を押し、管理コードを入力します。

画面表示が「管理モード」になれば準備完了です。作業後は右上の **ログアウト** を押してください。管理コードはブラウザを閉じても保持されません。

## 1. CSVをアップロードする

1. YouTube Studioで対象期間を「前週土曜〜金曜」にする。例: 7/4〜7/10のデータは7/14（火）中に取得する。
2. 詳細モードから、動画別の `コンテンツ別CSV` と日ごとの `日別CSV` をダウンロードする。
3. AI Directorの「CSVアップロード」を開く。
4. 管理モードにログインしていることを確認する。
5. 開始日（土曜）、終了日（金曜）を入力し、2つのCSVを選ぶ。
6. 「取込内容を確認」を押す。
7. 新規動画数、重複、日別CSVの欠け・期間外日付を確認する。
8. 通常は「別バージョンとして保存（推奨）」のまま取り込む。

同じCSVを2回選んでも二重登録されません。

## 2. 未確認動画を確認する

1. 「未確認動画」を開く。
2. その週に公開された動画だけ表示される。
3. 動画形式、企画ジャンル、出演メンバー、タグを確認する。
4. 間違った欄だけ修正する。
5. 複数動画を選び「選択した動画を確認済みにする」を押す。

確認済み動画は翌週表示されません。信頼度70%未満だけに絞ることもできます。

メンバーマスタを追加した後など、候補を作り直したい時は、対象動画を選択して「自動判定を再実行」を押します。確認済み動画の属性は再判定で変更されません。

確認済み動画を含む属性の修正は、「マスタ管理」内の「動画属性」から行えます。ここで変更した内容は自動判定で上書きされません。

限定公開・非公開の動画がある場合は、同じ「動画属性」の **公開設定** を `限定公開` または `非公開` にします。一度設定すると、以後の週次更新でも「今週の牽引コンテンツ」には入りません。CSVに公開設定列がある週は自動で反映されます。

## 3. AIレポートを確認する

1. 「AI Director」を開く。
2. 事実と集計結果を確認する。
3. 信頼度と対象週数を見る。
4. AI解釈と企画案は、根拠動画・数値がある場合だけ判断材料にする。

1週分の間は「参考値」「データ不足」と表示されます。これは不具合ではありません。

## 困った時

- 管理コードエラー: Cloud Run管理者に管理コードを確認する。
- 必須列エラー: コンテンツ別CSVに動画ID・動画タイトル、日別CSVに日付があるか確認する。
- 対象期間エラー: 土曜開始・金曜終了の7日間になっているか確認する。
- 文字化け: YouTube Studioから再ダウンロードする。
- 同一CSV: すでに取込済み。操作不要。
- 「保存が途中で中断」: 同じCSVをもう一度選び、「取込内容を確認」から再実行する。
- サイトは見えるが書けない: Cloud Runの環境変数とSheets共有権限を確認する。
- 週次レポートだけ古い: Apps Scriptのデプロイと `data/config.js` を確認する。
